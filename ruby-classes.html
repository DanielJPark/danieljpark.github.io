Blog Post #5 - Ruby Classes

March 20, 2016
In Ruby, classes work like a cookie cutter or template for objects. The basic idea and goal of a class is to create objects that behave in a similar fashion. This is done in a simple few steps.

First, the class must be defined and given a name. It will look like this:
class Name
Notice the name of the class is capitalized.
Then the class must be initialized and given arguements which looks like this:
def initialize(argument)
@argument = argument
end
Then define the argument.
def argument
@argument
end
end
