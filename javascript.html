Javascript vs Ruby
In Javascript Objects are very similar to Ruby Hashes. They both contain key:value pairs and are a way to store information. They are also both called upon the same way. With that said the syntax for them looks a little different. In Javascript the hash will require var and each key value pair is on its own line. In Ruby the key value pairs can be pairs with "=>" or ":".
